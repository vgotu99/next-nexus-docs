# 시작하기

## 설치

```bash
npm install next-nexus
# 또는
pnpm install next-nexus
# 또는
yarn add next-nexus
```

**필수 요구사항:** Next.js >= 14.2, React >= 18.2

## `NexusProvider`로 앱 감싸기

```tsx
// app/layout.tsx
import { NexusProvider } from 'next-nexus';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang='ko'>
      <body>
        <NexusProvider>{children}</NexusProvider>
      </body>
    </html>
  );
}
```

## API Definition 만들기

```ts
// src/api/productDefinition.ts
import { createNexusDefinition } from 'next-nexus';

export interface Product {
  id: string;
  name: string;
}

// 팩토리 함수(필수)
export const createApiDefinition = createNexusDefinition({
  baseURL: 'https://api.example.com',
  retry: { count: 1, delay: 1 },
  timeout: 5,
});

export const productDefinition = {
  list: createApiDefinition<Product[]>({
    method: 'GET',
    endpoint: '/products',
    server: {
      tags: ['products'],
      revalidate: 1800,
    },
    client: {
      tags: ['products'],
      revalidate: 300,
      cachedHeaders: ['x-total-count'],
    },
  }),
};
```

## 서버 컴포넌트에서 데이터 가져오기

```tsx
// app/products/page.tsx
import { nexus } from 'next-nexus';
import { productDefinition } from '@/api/productDefinition';

const ProductsPage = async () => {
  const res = await nexus(productDefinition.list);
  const products = res.data ?? [];

  return (
    <ul>
      {products.map(p => (
        <li key={p.id}>{p.name}</li>
      ))}
    </ul>
  );
};

export default ProductsPage;
```

## 클라이언트 컴포넌트에서 사용하기

```tsx
// components/ProductListClient.tsx
'use client';

import { useNexusQuery } from 'next-nexus/client';
import { productDefinition } from '@/api/productDefinition';

export function ProductListClient() {
  const { data, isPending } = useNexusQuery(productDefinition.list);
  const products = data ?? [];

  if (isPending && !data) return <div>로딩 중...</div>;

  return (
    <ul>
      {products.map(p => (
        <li key={p.id}>{p.name}</li>
      ))}
    </ul>
  );
}
```
