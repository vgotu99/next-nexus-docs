# 렌더링 위임

서버 컴포넌트 서브트리를 `<NexusSuspense>`로 감싸 위임을 활성화합니다.

## 조건
- GET Definition이며 `client` 옵션을 포함
- 클라이언트에 일치하는 캐시 키가 있고 `ttl > 0`
- 해당 서브트리에서 위임 컨텍스트가 활성화되어 있음

조건을 만족하면 서버는 fallback만 보내고, 클라이언트가 캐시로 즉시 렌더링합니다.

## 예시
```tsx path=null start=null
import { NexusSuspense } from 'next-nexus'
import { ServerComponent } from './ServerComponent'

export default function Page() {
  return (
    <NexusSuspense fallback={<div>로딩 중...</div>}>
      <ServerComponent />
    </NexusSuspense>
  )
}
```

## 주의사항
- 위임된 서브트리는 클라이언트에서 렌더링 가능해야 합니다(서버 전용 API 직접 사용 금지).
- 서버 로직은 서버 액션/Route Handler로 옮기세요.
