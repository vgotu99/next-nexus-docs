# 공통 관심사

읽기/쓰기 전반에 걸쳐 공통으로 중요한 주제들입니다.

## 인터셉터
- 요청/응답/오류 인터셉터를 한 번 등록하면 모든 nexus 호출에 적용됩니다.
- 인증 헤더, 텔레메트리, 공통 응답 가공을 중앙집중화하세요.

참고: [Interceptors](/ko/api/interceptors)

## 오류 모델
`isNexusError`로 신뢰성 있게 분기합니다.
```ts path=null start=null
import { isNexusError } from 'next-nexus/errors'

try {
  await nexus(def)
} catch (e) {
  if (isNexusError(e)) {
    console.error(e.code, e.response?.status)
  }
}
```
자세히: [오류 처리](/ko/api/errors)

## 캐시된 응답 헤더
- 서버가 노출하도록 정의한 응답 헤더는 엔트리와 함께 캐시되고 클라이언트에서 조회할 수 있습니다.
- 흔한 사례: 페이지네이션 합계(`x-total-count`), rate‑limit 정보 등.

설명/예시: [Definitions](/ko/api/definitions)

## 디버깅
- `NEXT_PUBLIC_NEXUS_DEBUG=true`로 개발 로그 활성화.
- START/SUCCESS/ERROR/TIMEOUT, HIT/MISS, SKIP/MATCH, DELEGATE, SET/UPDATE/DELETE 등 요청/캐시 이벤트가 표시됩니다.

참고: [/ko/debugging](/ko/debugging)
