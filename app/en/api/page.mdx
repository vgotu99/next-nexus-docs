# API Reference Overview

**Subpaths**

- `next-nexus` (Universal): `createNexusDefinition`, `interceptors`, `revalidateServerTags`
- `next-nexus/server` (Server-only): `nexus`, `NexusRenderer`, `NexusHydrationBoundary`, `withNexusHydrationBoundary`

- `next-nexus/errors` (Errors): `isNexusError`
